## 📖 Cách Sử Dụng

1. Clone repository:
   ```bash
   git clone https://github.com/quan02077/OOP---TH-OOP.git
   ```

2. Mở project bằng Visual Studio hoặc Visual Studio Code

3. Chọn folder bài tập muốn thực hành

4. Build và chạy các file `.cs` để xem kết quả
